Practice Exercise - Soulmate REST service

Soulmate is a start-up conceived with the idea to create an online match making platform.

Help Soulmate to set up their platform.

1.Complete all the endpoints for CRUD operations on Soulmate REST service where a site visitor should be able to:

a.) Add his/her profile on the platform

b.) Update his/her profile on the platform

c.) Delete his/her profile from the platform

d.) Search for: All soulmates registered with the platform Soulmates based on gender Soulmates based on gender and age

2.Use h2-console to view in-memory data

3.Add 2 endpoints to search user by name and search user by gender and age. (Understand @Query and parameter passing to @Query)
